AxeJS

Easy HTML5+JS UI, AJAX navigation and social networks integration, published as jQuery Plugin. For some things modern browser is required, old IE (below IE8) isn't supported and never will.

Info

It enables deep linking engine, so server-side scripts should check if it's XmlHttpRequest to response with full page or just it's contents.

The extended plugin contains logged user checks and some more features for editing (like buttons) and admin menus on the website. Also, logged in users get more debug information and warnings displayed directly when something goes wrong.

Also, jPlayer may be integrated into website, extended with Playlist plugin.
